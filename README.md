Tutorial
------
Recomended for python 2.7

Using git :

    $ git clone https://github.com/alishjoker/zzzzG


Python packages :

    $ pip install thrift==0.9.3
    $ pip install rsa
    $ pip install requests

Run alish :

    $ python alish.py
